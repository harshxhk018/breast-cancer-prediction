# Breast Cancer Prediction

Machine Learning project to predict breast cancer recurrence using patient clinical parameters.

## Dataset

- breast-cancer.csv

## Algorithms Used

- Logistic Regression
- Random Forest

## Results

- Logistic Regression Accuracy: 75%
- Random Forest Accuracy: 65%

## Files

- breast-cancer.csv
- breast_cancer_predict_18.ipynb
- breast_cancer_model.pkl
